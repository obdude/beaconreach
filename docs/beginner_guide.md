# Beginner Guide - Manual/Spreadsheet Path

1. Open `contacts/contacts.csv` in Excel or Google Sheets.
2. Choose a contact and read the pain_point column.
3. Open `templates/initial_contact.md` and fill in placeholders:
   - {{name}}, {{company}}, {{pain_point}}, {{asset_name}}, {{asset_link}}, {{sender_name}}
4. Send message via email or LinkedIn manually.
5. Log interaction in `value_history` and update `last_contacted`.
6. Schedule next follow-up in `next_action_date`.
7. Repeat using `followup_value.md` template as the next touchpoint.
8. Track engagement manually: responses, clicks, downloads.
