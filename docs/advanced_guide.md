# Advanced Guide - Python + AI + Workflow Path

1. Set up Python environment:

```bash
pip install pandas jinja2 openai
```

2. Edit contacts/contacts.json with your ABM contacts.
3. Edit templates in templates/ with Jinja2 placeholders.
4. Run workflows/python/beaconreach_demo.py to generate messages and log interactions.
5. Optional: integrate OpenAI API to auto-personalize messages.
6. Optional: integrate n8n or Zapier workflows for automated sending and logging.
7. Track KPIs using outreach_dashboard.py
